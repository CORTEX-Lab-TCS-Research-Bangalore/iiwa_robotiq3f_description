# iiwa_robotiq3f_description
This is a MJCF asset for simulating IIWA for Robotiq 3 F Adaptive gripper in Mujoco.

***NOTE*** : 
The original asset of Kuka LBR IIWA 14 was taken from [Mujoco Menagerie](https://github.com/google-deepmind/mujoco_menagerie) and the Robotiq 3F asset was adapted from [Robotiq](https://github.com/ros-industrial-attic/robotiq/tree/kinetic-devel).
